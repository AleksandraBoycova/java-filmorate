# java-filmorate
Template repository for Filmorate project.

# Проект базы данных для приложения 
![](https://github.com/AleksandraBoycova/java-filmorate/blob/database-project/QuickDBD-export.png)

В таблице **Film** хранятся данные о фильмах. 

Таблица **User** хранит информацию о пользователях. Поля *Email* и *Login* уникальны.

В таблице **Genre** хранятся возможные названия жанров фильмов. В таблице **Film** есть внешний ключ к таблице **Genre** через *GenreID*.

В таблице **MPA** хранится рейтинг Ассоциации кинокомпаний (название и пояснение). В таблице фильм есть внешний ключ к таблице **MPA** через *MPAID*.

Чтобы хранить информацию о рейтинге фильма существует таблица **FilmLike**. Она хранит в себе информацию о том, какой пользователь, к какому фильму поставил лайк. В этой таблице есть 2 внешних ключа: к таблице **Film** через *FilmID* и к таблице **User** через *UserID*.

Чтобы хранить информацию о дружбе пользователей существует таблица **Friendship**. Информация хранится в виде ID пользователей, которые дружат и статуса дружбы: если пользователь отправил запрос на добавление в друзья, но не получил подтверждение статус не подтвержденный, а если получил, то подтвержденный. Эта таблица имеет 2 внешних ключа к таблице **User** через *UserID*.

# Примеры запросов для основных операций приложения: 
```
SELECT *
FROM Film
WHERE FilmID = 1(найти фильм с ID 1);
```

```
SELECT *
FROM User
WHERE UserID = 1(найти фильм с ID 1);
```
